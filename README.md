# Shoppia
A development of Shoppia E-commerce website using html, css, React, Tailwindcss, ExpressJS and Node.js

## Prequisite
- Your browser version should be up to date for better experience

## Features
- Home page
- About page
- Signup page
- Login page
- Contact page
- Checkout page

## Tech/framework used
- React
- Css
- TailwindCss
- Nodejs
